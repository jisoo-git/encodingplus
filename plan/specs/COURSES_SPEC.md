## COURSE 01 — 입시 단기특강

| 항목 | 내용 |
|------|------|
| 전형 | 특별전형 + 일반전형 (모두) |
| sub | 특별전형과 일반전형 모두 도전 |
| 수강료 | 73만원 / 월 · 4회 기준 |

### 상세 설명
두 전형을 모두 도전하여 합격률을 높이는 전략입니다.

특별전형 대비부터 일반전형 대비까지 16주에 걸쳐 종합적으로 대비합니다. 2027학년도 과정은 7월 18일에 개강합니다.

### 수업 구성

| 영역 | 내용 |
|------|------|
| 코딩/콘텐츠 | 실적물 생산, 이론 교육 |
| 적성고사 | 이산수학, 정보학습, 실전모의고사 |
| 입시논술 | 실적설명서, 자소서, 면접대비 |

### 수업 시간

| 요일 | 시간 | 형태 |
|------|------|------|
| 매주 토요일 | 12:00 – 18:00 | 대면 6시간 |
| 매주 수요일 | 오후 10시 또는 11시 | 비대면 1시간 · 보충 불가 |

## COURSE 02 — 일반전형 특강

| 항목 | 내용 |
|------|------|
| 전형 | 일반전형 |
| sub | 일반전형 집중 전략 |
| 수강료 | 48만원 / 월 · 4회 기준 |

### 상세 설명
일반전형은 내신과 소질적성검사가 핵심입니다. 기본 정보 역량과 적성고사 대비 정도에 따라 합격이 좌우됩니다.

코딩 수학과 정보 교육을 중심으로 소질적성검사를 대비하고, 면접을 함께 준비합니다.

### 수업 구성

| 영역 | 내용 |
|------|------|
| 적성(수학) | 이산수학을 바탕으로 수학 수업 진행 |
| 정보(컴퓨터) | 컴퓨터 교과 내용과 상식 차원의 정보 수업이 진행 |
| 입시논술 | 인성 면접 대비 |

### 수업 시간

| 요일 | 시간 | 형태 |
|------|------|------|
| 토 또는 일요일 | 15:00 – 18:00 | 대면 3시간 |
| 매주 수요일 | 오후 10시 또는 11시 | 비대면 1시간 · 보충 불가 |

| 전형 | 설명 |
|------|------|
| 특별전형 | 실적물·소질적성검사·자기소개서·면접으로 평가합니다. IT 관련 실적 활동이 있는 학생에게 유리하며, 실적물 제작과 이론·논술을 종합 대비합니다. |
| 일반전형 | 내신과 소질적성검사(C언어·정보소양·논리적사고)로 평가합니다. 코딩을 처음 시작하는 학생도 충분히 준비할 수 있습니다. |
| 공통 사항 | 디미고는 전기학교로 전국 1곳만 지원 가능합니다. 두 전형에 동시 지원할 수 없으므로 지원 전형을 신중히 결정하세요. |
